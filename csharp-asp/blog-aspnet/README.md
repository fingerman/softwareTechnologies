## DEMO BLOG C# ASP.NET
================

Start with the repo:
```bash
$ git clone https://github.com/fingerman/softwareTechnologies.git
```
##### C# ASP.NET Demo Blog:
```
$ cd csharp-asp/blog-aspnet/
```

Check for SQL Server LocalDB Installation  
**cmd -> sqllocaldb**  
If missing ? download and install SQL Server LocalDB  
just run ```Blog.sln``` Visual Studio or your prefered IDE 


The DEMO BLOG App is preconfigured. The DB should be automatically-generated **App_Data** when running.   
If not just configure an ASP.NET Web Application with VS:
- choose MVC and untick the "Host in the cloud" checkbox if any  
- change the authentication method to “Individual User Account”
